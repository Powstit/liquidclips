## 🎯 Priority sequence — what to ship next, in order

The single biggest lift toward $100M look = the ❌ rows that touch the daily-user path.

**Sprint A — kill the staged feeling**
1. ⌘K Command Palette (the headline pro affordance)
2. 7 more unique clip thumbnails (so the grid stops looking like 5 actors)
3. Real platform brand icons in `iconography/platforms/` — Daniel's exact ask
4. Single bold `+` button replacing tiny action icons — Daniel's exact ask
5. Empty-state hero with crafted "what are we making?" copy
6. Loading skeleton (9-card shimmer, real, not generic)

**Sprint B — every state crafted**
7. Bake-failed strip + retry CTA
8. Offline / sidecar-restarting state
9. Free-tier watermark preview
10. Toast host + 4 toast variants (info / success / warn / error)
11. ConfirmDialog branded

**Sprint C — voice + sound**
12. Empty-state copy across every room ("what are we making?", "no channels routed yet — pick one")
13. Error copy ("we lost contact with the sidecar — auto-restart in 4s")
14. Sound design (5 cues)
15. Keyboard shortcut overlay (`?` to invoke)

**Sprint D — atmosphere**
16. Motion register expansion (fast 60ms, medium 240ms, slow 1.6s, ambient 5s — currently only medium)
17. Per-clip baking state with mini progress bar inside the card
18. Confetti reel on first publish
