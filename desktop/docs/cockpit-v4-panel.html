Panel-design-lens budget

  • Scan-units at rest: CLIP chip · focusnav · ⓘ · counters · ⋮ + 5 module headers = 10. Modules vertically separated, scan groups well — fits.
  • Action budget: 1 primary (CTA, morphs Schedule ↔ Publish) + 2 secondary (Routes & Schedule, ⋮ menu). ✓
  • Vocabulary budget: Familjen Grotesk + JetBrains Mono (2 type families). Fuchsia primary + cyan info + amber warn + red error. Rounded-md pills + rounded-lg panels (2 shapes). Solid borders only. ✓
  • Repeat budget: zero duplicated actions ✓
  • Pending state: teal bake strip with sweep animation + Cancel ↑. Tiles + rx-config dim, CTA disabled. Mirrors import-pending pattern. ✓
  • Error state: red error strip + Retry. Tiles re-enabled. CTA flips to Publish now (since bake failed, schedule+1h would publish a clip with no reaction). ✓
  • Card mirror: teal bake-strip and red error-strip on the cards mirror the cockpit's vocabulary — single visual language across grid + panel.
