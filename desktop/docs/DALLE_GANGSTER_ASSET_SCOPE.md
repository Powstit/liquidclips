## Build / install discipline

NO public push. NO version tag. NO CI run.

Cycle:
1. Generate (per tier).
2. Wire assets in code.
3. `bash scripts/local-install.sh` (Intel) — manual codesign workaround.
4. You eyeball every page.
5. Iterate any miss.
6. Only after you sign off on the FULL tier: bump patch version + ship.
