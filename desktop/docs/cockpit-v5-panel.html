Honest scan-unit count at rest

  • Status strip (6 widgets): CLIP·V70 chip · focusnav · ⓘ · counters · Schedule pill · ⋮
  • Primary row (3 modules): Caption-with-style · Reaction tiles · Deploy CTA — each acts as one scan group
  • Secondary row at rest (2 units): Source pill · Advanced ▸ toggle
  • Total scan groups: 11 — still over the 7 budget but well under v4's ~14. Acceptable for a working panel where Reaction + Caption + CTA must all be visible.

What got cut from v4 → v5

  • Style module — merged into Caption as a single swatch + style-tag inline ✓
  • When module — collapsed into the status-strip schedule pill (click opens the 4-chip popover) ✓
  • Score gauge module — collapsed into a status-strip V70 pill ✓
  • Routes & Schedule module title — Routes is now the secondary button under the Deploy CTA ✓
  • Audio + Offset visible at rest — hidden behind Advanced ▸ disclosure (cuts ~5 units when not needed) ✓

When the secondary row appears

  • Layout set → shows source pill + Advanced ▸ toggle
  • Advanced clicked → audio segmented + offset slider appear
  • No layout → single one-line prompt: "Pick a layout above…"
  • Baking → replaced entirely by teal pending strip (with Cancel)
  • Failed → replaced entirely by red error strip (with Retry)
