v6 vs home language

  • Black background (matches home — no gradients, no ink-2/ink-3 layering)
  • Dashed fuchsia corner brackets frame every cockpit-tile + every card (matches home's Create/Import/Thumbnails/Script tiles + the studio HUD framing)
  • Large fuchsia line icons at top of each tile (matches home: sparkles, layers, image, scroll → ours: speech, layers, calendar, rocket)
  • Lowercase sans-serif headlines per tile (caption, reaction, schedule, publish)
  • Spaced mono uppercase sublabels with optional fuchsia bold highlight (matches home's PASTE A LINK · DROP A VIDEO)
  • 4 tiles in primary row — same rhythm as home (Create · Import · Thumbnails · Script → Caption · Reaction · Schedule · Publish)
  • Reaction tile is wider to host the 8 mini-diagram layouts directly inside
  • Publish tile = primary CTA in the home's primary-tile style (live = fuchsia halo, now = amber halo)

Scan-unit count

  • Status strip: 6 (clip-pill · v-pill · focusnav · health · counters · sched-pill · ⋮ = 7 widgets but all monoline so reads as one band)
  • Primary row: 4 cockpit-tiles as scan groups (Caption · Reaction · Schedule · Publish)
  • Secondary row at rest: 2 (source · Advanced toggle)
  • Total: 12 — within working budget for a real cockpit. The 4-tile primary row matches the home's exact rhythm so the user's eye knows where to land.
