# Liquid Clips Marketing

Next.js marketing site for `liquidclips.app`.

## Local

```bash
npm install
npm run dev
```

## Required Launch Wiring

- Point `liquidclips.app` at the Vercel project.
- Set `NEXT_PUBLIC_DOWNLOAD_DMG_URL` once the notarized GitHub Release DMG exists.
- Replace the demo placeholder link with Daniel's Loom/screen-capture demo when recorded.
